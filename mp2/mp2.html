Overview and Usage

This project was a chance to explore P5js. I created 6 different images that each have a different functionality. I decided to have a Christmas Tree theme that remains the same for each of the 6 drawings and adds a new functionality in each drawing.

  • Static Image: This image is a stand alone Christmas Tree
  • Random Image: This image radnomizes the size and color of each ornament on the tree.
  • Loop Image: This image features a spinning star on the tree.
  • Interactive Image: The user can click anywhere on the tree to add a new ornament.
  • Audio Image: The user can click on the screen to play a popular Christmas song.
  • Audio Visual Image: The user can hover over the ornaments on the tree to play Christmas songs.

Development Process

  • I created a statis basic Christmas tree first, inclusive of functions for each part of the tree so I could make a consice draw fcuntion and make any changes efficiently with the functions.
  • For the random drawing I made multiple for loops to randomize the radius of each circle drawn as well as 3 random rgb values to pass into my ornament function.
  • To create an infinite loop, I looked at multiple online sources to find a way to create a spinning star. I did so finally by using the rotate functon avaiable in p5js.
  • Initially for the interactive picture I wanted to decorate the tree by drawing multiple line but ended up making the user click and add ornaments to the tree. I did so my creating a list and a circle function that tracked the users mouseX and mouseY coordinates on a click and added it to a draw list that then drew circles where a user clicked.
  • I downloaded a mp3 of a popular Christmas song and played a song on a mouse click and off on another mouse click. I had to incorporate the sound library for p5js which was different than what I had done before.
  • For combining an audio visual I had to search and examine a couple exampled of how to make music play on a hover over a certain shape. This was done so by calculating the distance of the users mouse to the edge of the shape you're on. If its within the shape a function to play the music will be run and once the distance is greater than the shape, the music will stop.

Issue Deep-Dive

An issue that I encountered was in my audio visual drawing where I was hovering over an ornament and I was getting a distorted version of the song it should have been playing. It sounded like it was playing over it self over and over. After investigating I realized the problem to be that I was playing my song on a hover variable status in the draw loop. This means that my song was continously playing becuase the hover variable status was always true and the song would keep playing over and over. While I still wanted to keep my song playing in the draw loop I needed some sort of conditional to track whether or not the song was already playing. After messing around with a couple ways and places to put this variable, I opted for a global variable that would keep track of whether or not the song was already playing. It took a couple tries to figure out where in my code I should update this variable but I figured it out.
