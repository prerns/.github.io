Final Project MVP Writeup

Overview and Usage

This final project prototype is intended to serve as a MVP of my final project. I have created an artboard on p5 that currently is able to save and clear the artboard as well as change the artboards background color and draw lines. This is the first iteration of my final.

Artboard Prototype

Development Process

  • Created a canvas to be my artboard
  • Implemented a slider using the createSlider function that changes the background color of the canvas
  • I created a button that implemented the save method to save the artboard to the users computer
  • I created a clear button and worked through different methods of clearing the artboard and looked at many examples online
  • I created a button to draw lines and implemented a method to draw lines on mouse press that I had used in my sketches.

Issue Deep-Dive

One issue I had was trying to create a function that would clear the artboard. after looking into the clear and remove methods from p5 I realized that those methods would not perform what I wanted. After looking at some examples online, I tried to create a function that would draw a rectangle the size of the canvas over the current artboard as a way of creating a new blank canvas that was filled with the same background color that was selected. However, when I tried to draw lines again on this new artboard nothing would show up. After some trouble shooting, I figured out that the array that was storing the lines was not being cleared when the clear button was being called so the lines were being drawn on the layer under. Then I tried another method of just setting my lines array to an empty array to "clear" the artboard instead of drawing a new canvas over the existing one this worked!
