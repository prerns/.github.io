Final Project Proposal

Project Description

This project will be a P5 artboard that allows the user to draw and manipulate the canvas in muliple ways and then save and download the art they created.

Iterations

Prototype Version

The prototype version of this project will include one shape that the user can draw with and the ability to change the color of the background, the user can also clear the artboard and save it.

Final Submission

The final version of this project will allow the user to select between 4 different shapes to draw from as well as change the color of the background. The user will be able to switch between the shapes and clear the background as well as save and download the art.

Development Process

  • Create a blank canvas
  • Have a mouse click create a cicrcle on the screen. Once complete, create a button that initiaties this functionalitiy.
  • Create a slider that changes the background color of the canvas using the p5 methods.
  • Create a button to clear the canvas using the p5 methods.
  • Create a download button using the p5 methods. This is my MVP
  • Create buttons that act similarily to the circle button for a line shape and a rectangle shape.
  • Match the UI with css to my desired look. This is my final.

Unknowns

I don't know the methods for creating a slider or downloading and saving but I know they exist within p5 documentation. I also don't know how to create functioning buttons on p5.
